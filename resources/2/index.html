    Godot TIE

    Éditeur de dialogue
    Text Interface Engine est un plugin qui permet d'afficher du texte comme dans les jeux de type RPG.
    2.1 3.0 3.1 Dialogue Éditeur Plugin

    GodotSteam

    Steam SDK
    Module C++ qui va vous permettre de brancher Godot Engine avec le SDK de Steam. Nécessaire si vous voulez publier votre jeu sur la plateforme Steam.
    2.1 3.0 3.1 Steam Module

    Godot OpenVR

    OpenVR
    Plugin GDNative qui ajoute le support d'OpenVR pour godot Engine.
    master 3.0 3.1 VR Module

    Godot Mixing Desk

    Son interactif
    Solution audio complète, permet de générer des sons procéduralement, et d'avoir une musique adaptative/procédurale en ajoutant quelques nodes accompagnés de quelques lignes de code
    3.1 Plugin

    Godot Lua

    Godot Engine
    Version de Godot Engine supportant le Lua. C'est toujours en développement donc il vaut mieux ne pas l'utiliser pour de la production
    master 3.2 Lua

    Godot Jigglebones

    Animation procédurale
    Ce plugin ajoute les parties moles a vos squelettes 3D. Cela permet de d'avoir des animations procédurales plus réalistes.
    3.0 3.1 Jigglebone Animation Procédural Plugin

    Godot Google Service

    Google Service
    Module C++ permettant d'utiliser les services offerts par Google sur Android.
    2.1 3.0 3.1 3.2 master Android Google Services Module

    Godot ANL

    Bibliothèque Accidental Noise
    Il s'agit un module C++ qui fait le lien entre la bibliothèque Accidental Noise et Godot Engine. Une fois compilée, on peut s'en servir avec du GDscript.
    3.1 Module Noise Shader

    GodotAds

    Publicité Android
    Module permettant de gérer les publicités pour Android avec Godot Engine. Il gère notammenet admob, adcolony, chartboost, vungle, mopub et unity_ads
    2.1 3.0 3.1 Android Module

    GDTwitch

    Twitch
    Permet d'intéragir avec le chat de Twitch
    3.1 Twitch Plugin

    GUT

    Utilitaire
    Godot Unit Test est un utilitaire permettant d'écrire des tests pour votre jeu. Cela permet d'écrire des tests en GDScript pour tester du code GDScript
    2.1 3.1 3.2 Plugin

    Godot Voxel

    Bibliothèque Voxel
    Module écrit en C++ permettant de créer des volumes dans Godot Engine. Voxel c'est la contraction de Volumetric Pixel.
    3.1 Voxel C++ Module
